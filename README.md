## Endpoints

`GET /blog`

- yml 목록 데이터에서 유튜브 목록을 읽어옵니다.
- 플랫폼에 따라 rss주소를 만듭니다.
- rss주소별 피드를 뽑아서 json으로 저장합니다.

`PUT /blog`

- 기존에 있던 collection에 바뀐 부분을 업데이트합니다.

`POST /blog/feed/save/to/file`

- yml 목록 데이터에서 블로그 목록을 읽어옵니다.
- 플랫폼에 따라 rss주소를 만듭니다.
- rss주소별 피드를 뽑아서 json으로 저장합니다.

`POST /blog/feed/save/to/mongo`

- mongodb에 yml 목록 데이터를 입력합니다.
